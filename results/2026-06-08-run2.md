### 4. `test_32_3_membership_type_updates_pricing`
**Classification:** Test code bug (jQuery CDN dependency)
**Error:** `AssertionError` â€” `#pricingSummary` did not appear.
**Root cause:** The pricing JS is triggered via a jQuery `change` event handler on
`#membership_type_id`. When jQuery CDN load is slow or fails, `select_option()` fires
a native DOM change event but jQuery's delegated handler never fires.
**Fix applied:** After `select_option()`, call `page.evaluate()` to dispatch a native
`Event('change', {bubbles: true})` which both native and jQuery listeners receive.
Committed in `6ac52bd`.
